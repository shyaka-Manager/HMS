# Backend - Hospital Booking API

Express + mysql2 + cors for MySQL (XAMPP).

## Prerequisites

- Node.js 18+
- XAMPP (MySQL running)

## Environment Variables

Create `.env` in this folder:

```env
PORT=3000

SQL_HOST=localhost
SQL_PORT=3306
SQL_DATABASENAME=hospital_booking
SQL_USERNAME=root
SQL_PASSWORD=

FRONTEND_URL=http://localhost:5173
API_BASE_URL=http://localhost:3000
```

## Install and Run

```bash
npm install
npm run start static server
npm run dev dynamic server
```
